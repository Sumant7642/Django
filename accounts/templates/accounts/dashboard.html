User Manual

The user will register into the application by providing all the necessary details and therefore he can log in into the application using username and password and the user can upload the files to server and share with the other registered users. He can also view the files uploaded by him . The key generated by the uploaded user will be shared personally. Using that key, he can download the file and view the information.
